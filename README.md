# service-weaver-demo

## Introduction

This is a demo service build with [Service Weaver](https://serviceweaver.dev/) to showcase a simple modular monolith application.

## How to run?

### Requirements

- Go
- GNU Make
- SQLite3

### Run

First, initialize the database with:

```bash
make db
```

Then, run the application with:

```bash
make dev
```
